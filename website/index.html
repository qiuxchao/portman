How It Works

Three steps. Zero config.

1

Open PortMan

All listening ports are scanned automatically on launch. No setup required.

2

Find the port

Search or scroll to find the port. See exactly which process is using it and how it was started.

3

Kill or copy

Click the kill button to terminate the process, or copy the port number to your clipboard.
